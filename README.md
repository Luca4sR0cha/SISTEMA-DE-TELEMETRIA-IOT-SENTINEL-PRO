# Dashboard Industrial SCADA - Indústria 4.0

## Descrição

Este projeto é um **Dashboard SCADA** para monitoramento e controle de processos industriais, simulando conceitos da **Indústria 4.0**. Ele permite visualizar dados de sensores, acompanhar status de máquinas e interpretar indicadores de produção em tempo real.

O objetivo é fornecer uma interface visual que integra dados industriais, simulando uma fábrica conectada e inteligente.

---

## Funcionalidades

- Monitoramento de dados de sensores em tempo real.
- Visualização de status de máquinas e processos.
- Gráficos e dashboards interativos.
- Estrutura modular para fácil expansão.
- Simulação de processos industriais típicos da Indústria 4.0.

---

## Tecnologias utilizadas

- HTML / CSS / JavaScript (front-end)
- Node.js / Python (para simulação e backend, se aplicável)
- MQTT / WebSockets (para comunicação em tempo real)
- Bibliotecas gráficas para dashboards (Chart.js, D3.js ou similares)
